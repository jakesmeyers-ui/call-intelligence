Step 1 of 5
Opening & Reassuring Statement
The first 10 seconds determine whether the customer stays or goes.
Did the agent use the verbatim opening script?
"Thank you for calling reservation desk/counter, an independent travel agency. Hi this is [Name], with whom do I have the pleasure today?"
Did the agent clearly identify themselves AND the company?
Customer must know who they're speaking with and that this is an independent travel agency — not the hotel.
Did the agent use a reassuring transitional statement?
"You called the right place, I'll be happy to help you with your booking at [hotel]"
Did a RUTH question come up? (customer asked if agent is/works for the hotel)
RUTH = "Are you the hotel?" / "Is this the front desk?" / "Are you affiliated with the hotel?" etc.
If RUTH came up — what did the agent say?
Correct: "No, you called [brand from opening], an independent travel agency." — must use the same brand they opened with.
Step 2 of 5
Probing & High-Impact Questions
Did the agent gather enough info to personalize and get ahead of objections?
Did the agent ask for check-in and check-out dates?
Did the agent ask for number of rooms and guests?
Did the agent ask for room type preference?
"Do you prefer one bed or two beds? Any preference for room type?"
Did the agent ask at least one High-Impact Question (HIQ)?
"What brings you to the area?" / "What amenities are you looking for?" / "What's your deal breaker?" / "What made you choose this hotel?"
Did the agent use HIQ answers to personalize the offer and anticipate objections?
Step 3 of 5
Features, Benefits & Offer
Was the offer built correctly and presented with confidence?
Did the agent present features that matched what the customer said they needed?
Benefits must be tied to probing answers — not generic amenity lists.
Did the offer include bed type, at least 2 amenities, cancellation policy, and price?
Formula: Bed type + two amenities + cancellation policy (if available) + price
Did the agent frame the rate as exclusive or promotional?
Did the agent close IMMEDIATELY after the offer — without pausing or waiting?
The close must happen right away: ask for name → email → card. No hesitation.
Step 4 of 5
Sense of Urgency
Urgency is not a rescue tool — it must be planted before the offer, every single call.
🚨 Did the agent apply urgency DURING the offer — not just when the customer objected?
4 triggers: far-out date ("great you're calling now, books up fast") / last-minute ("oh my, that's last minute, you're through to the right person") / location ("you know how [city] can be this time of year") / visit reason ("with that tournament, lots of people are grabbing rooms"). Must be proactive — not saved for objections.
Did the agent give a specific reason for urgency?
Weekend / holiday / high season / events in the area / last-minute / one of few rooms left / popular destination
Did the agent mention Free Cancellation and Exclusive Rate as part of the Holy Trinity?
Holy Trinity = Sense of Urgency + Free Cancellation + Exclusive Rate — all three ideally on the offer. Exception: same-day check-in — free cancellation is typically not available, so use Urgency + Exclusive Rate only
How was the agent's tone and acting?
Step 5 of 5
Objection Handling — The ARC Method
Acknowledge → Respond → Close. How did the agent handle resistance?
Was there an objection on this call?
A — Did the agent ACKNOWLEDGE with empathy first?
"I understand…" — validate before pushing back.
Did the agent dig for the REAL reason behind the objection?
"What's stopping you from making the reservation?" — most objections are shields.
R — Did the agent RESPOND with the Holy Trinity / a real solution?
"However…" — Urgency + Free Cancellation + Exclusive Rate. Not just restating the price.
When cornered, did the agent Double Down?
State Free Cancellation (explain it fully) + Urgency + Exclusive Rate one by one, recap, then close.
C — Did the agent CLOSE immediately after resolving the objection?
"So tell me…" → straight to booking. No re-opening the same topic.
Did the agent avoid offering a callback as the first option?
Only 1 in 10 callbacks become a sale. Push to the limit before offering it.
